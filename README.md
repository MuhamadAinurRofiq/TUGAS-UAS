1. Masuk ke login, jika ok lanjut ke pemilihan progam. jika gagal  akan kembali mengulang login
Masuk ke pemilihan progam. pilih progam yang akan dijalankan 
Jika memilih 1 maka akan menjalankan progam gaji
Setelah progam gaji dijalankan akan muncul pilihan ingin tambah data atau tidak
Jika ya makan akan mengulang ke progam penggajian jika tidak akan lanjut ke menu pilihan progam
